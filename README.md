#### 1. why pytorch?

(1) 파이토치 장단점

 장점 
 - numpy와 매우 비슷한 문법
 - 동적으로 back-propagation 경로 생성 가능
 - 너무나도 훌륭한 documentation
 - 난이도에 비해서 자유도 높음

단점
- TensorFlow에 비해서 낮은 상용화 지원
- TensorFlow는 구글에서 pytorch는 페이스북에서 만듬 (단점은 아니고 특징)
